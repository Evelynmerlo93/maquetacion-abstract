📄 Abstract Help Center – Layout Project

📌 Description

This project is a layout implementation of the Abstract Help Center, developed as a practice exercise at IT Academy.

The goal is to reproduce a real UI design using semantic HTML, modern CSS techniques, and a mobile-first approach.

🎯 Project Objectives
Build a semantic HTML structure
Apply CSS using Flexbox and Grid
Use a mobile-first workflow
Work with relative units (rem, %)
Improve basic accessibility
Recreate a real-world layout design
🧱 Project Structure

The layout is divided into four main sections:

Header → main navigation bar
Hero → title and search input
Cards → content sections using CSS Grid
Footer → links organized in columns
🛠️ Technologies Used
HTML5
CSS3
Flexbox
CSS Grid
Media Queries (responsive design)
📱 Responsive Design

The project follows a mobile-first approach:

📱 Mobile: single column layout
💻 Desktop: multi-column layout using CSS Grid
♿ Accessibility

Basic accessibility improvements were applied:

Semantic HTML elements (header, main, section, footer)
Proper heading structure (h1, h2, h3)
aria-label used in the main section
